# BFE BTU Calculator
*A sophisticated HVAC calculation tool built with modern React architecture*

## Tech Stack
- **React 18** with TypeScript for type-safe component development
- **Zustand** for efficient global state management
- **Bootstrap 5** with custom SCSS for responsive design
- **Vite** for lightning-fast development and optimized builds
- **i18next** for internationalization (English/French support)

## Key Features
- Real-time BTU calculations with instant user feedback
- Component-based architecture with separation of concerns
- Responsive design optimized for desktop and mobile devices
- Multi-language support with dynamic language detection
- Clean, maintainable codebase following React best practices

## Architecture Highlights
- **State Management**: Zustand stores for scalable state handling
- **Component Structure**: Modular components with clear responsibilities
- **Type Safety**: Full TypeScript implementation for robust code
- **Performance**: Optimized build pipeline with Vite
- **Accessibility**: Semantic HTML with Bootstrap accessibility features
