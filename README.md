# BFE BTU Calculator
Estimate the number of BTUs needed to heat or cool a room or house based on size, insulation, and more.

## Tech Stack
React, TypeScript, Zustand (Global State Management), Bootstrap, SCSS

## Features
- Calculates BTUs based on room size and insulation
- Dynamic, real-time updates
- Responsive design for desktop and mobile
